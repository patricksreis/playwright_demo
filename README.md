# playwright

1. Vá para o caminho do playwright

   ```
   cd playwright
   ```

2. Instale as dependências

   ```
   npm install
   ```

3. Instale os navegadores

   ```
   npx playwright install
   ```

4. Execute os testes

   ```
   npm test
   ```
